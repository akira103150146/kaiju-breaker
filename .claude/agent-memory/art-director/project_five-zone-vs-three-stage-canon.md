`design/feedback/2026-07-02-改進意見與劇情草案.md` §B proposes a **5-zone** stage
structure (墜落點・廢棄港灣 / 鏽蝕市街 / 污染濕地・生體巢穴 / 軌道電梯遺構 / 魔物母巢・核心)
with 5 new boss names (甲殼中型魔物 / 高速掠食型 / 孵化母體 / 機械寄生型 / 巨型母體 Kaiju).

`design/gdd/stage-system.md` §G already defines a **confirmed 3-stage** canon
(S1 Reef Outpost/CARAPEX, S2 Abyssal Rift/LACERA, S3 Voltage Spire/VOLTWYRM),
each with its own background mood description (cold-leaning palettes, warm-but-
desaturated accents) — see `art-bible.md` §5 for the 3 kaiju's canonical colors.

**Why:** The feedback doc explicitly flags itself as unconfirmed backlog
("尚未成為正式 GDD... 待你確認") — the 5-zone draft has NOT been reconciled with
the existing 3-stage/3-kaiju canon. It's unclear whether the 5 zones are a
renumbering/expansion of the existing 3, or an entirely separate narrative pass.

**How to apply:** Any art-director work that touches stage/zone themes (backgrounds,
boss color direction, per-zone palettes) must treat the 5-zone draft as
*aspirational reference only*, never as confirmed canon, until the director
resolves this mapping. Flag it as an open question rather than silently picking
one. See `design/art/scrolling-background-parallax.md` §I/§J for how this was
handled for the parallax background spec — reuse that pattern (worked examples
against the confirmed 3-stage canon; the 5-zone draft only as a labeled "未確認"
reference table).
