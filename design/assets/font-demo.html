選一個科技感像素字型

點任一字型卡 → 上方 HUD 立即用該方向重繪。字是用 canvas 以真實 16px 像素網格渲染的（1-bit 去鋸齒），所以你看到的是介面實際的像素質感與版面。誠實聲明：這些是「方向預覽」，不是各字型的真實字面 —— 真實 TTF（尤其繁體 CJK 字面）無法在網頁沙盒內嵌入。選定方向後，把該 TTF 丟進專案，我就能在 Unity 裡出真實對照截圖。

選好了怎麼看真實字面

  1. 1 從下載連結抓該字型 TTF（16px 版本）。
  2. 2 丟進 Assets/_Project/Art/Fonts/。
  3. 3 跟我說一聲 → 我在 Unity 裡把 HUD/選單用該字型渲染，出真實截圖對照，你再拍板。
  4. 4 定案後我建 TMP bitmap font asset（point filter / 整數縮放），接進正式 UGUI HUD。
