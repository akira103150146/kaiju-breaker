## 使用說明

**最快路徑（推薦）**：
1. 先把三隻的 **§1.0 / §2.0 / §3.0「全身參考圖」** 一次生成，確認整體造型方向對。
2. 滿意後，對每張全身圖下**編輯指令**逐件切出：「把這隻的左大顎單獨抽出來、透明背景、其餘留白」→ 各部件風格一致、接合邊自然吻合。
3. 或直接用各部件的獨立 prompt（§x.1 起）分開生成。

**每個新對話先貼一次 → 全域風格設定：**
> 我要做一款復古街機像素風垂直捲軸射擊遊戲的 boss 素材。請生成**單一遊戲精靈（single game sprite）**、**透明背景**、**16-bit 街機像素風、大顆明顯方形像素、無抗鋸齒、無平滑漸層、硬邊緣、扁平色塊**。每個部件是要在引擎裡**組裝**成完整 boss 的**獨立元件**：畫成**完整、邊緣乾淨、輪廓封閉**的單一部件，整個完整入框不被裁切，連接身體那一側畫成**平滑乾淨的接合邊**。避免：撕裂/斷裂/破碎/鋸齒斷口/殘肢般的邊緣（破損態除外）。

## 三條鐵則（別踩）

1. **CARAPEX / LACERA = 活體血肉甲殼生物** — 暖色 / 病態色，有機、濕潤、病態質感。「殼」是**活的幾丁質生物外殼**（像螃蟹殼、甲蟲鞘翅），不是金屬裝甲。
2. **VOLTWYRM = 純能量 / 等離子生命體** — 不是血肉、也不是機械。層疊的光、能量弧、等離子。
3. **玩家 = 科技 = 冷色幾何**。巨獸**絕對不能有**金屬、機械、機甲、機器人、齒輪、電路、乾淨幾何硬表面。
   - **唯一冷色例外**：VOLTWYRM 的**護盾**（深紫藍 `#503090`），刻意冷色以免被誤讀成敵彈；剝殼後轉暖。其餘所有部位一律暖色 / 病態色。

# 3. VOLTWYRM 熾蛇（#03 能量系 · L4 展示 Boss）

**造型**：**垂直蛇柱**，由上到下堆疊的分節，頂端核心 + 兩側護盾。占屏寬 ~25%、高 70–80%（三隻裡最高最窄）。
**配色**：核心極亮白黃 `#FFFFF0`（全遊戲最亮點）· 能量弧黃 `#FFE860` · 橙紅邊 `#FF9020` · **護盾冷紫藍 `#503090`／內層 `#302080`（唯一冷色例外）**· 1px 暗暖黑描邊 `#1A0800`。
**⚠️ 這隻是「能量/等離子」不是血肉**：純光、能量弧、等離子，無有機肉質、無甲殼、無金屬機械。

### 3.0 全身參考圖
